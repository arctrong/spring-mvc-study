About this section

This section is intended for discussion on different ways of embedded server setup with Spring MVC.

The topic doesn't look to be simple and straightforward. There are quite many materials on the Internet but they all probably cover the simplest use cases and contain very little or no descriptions and explanations. Some of the materials are listed in [23], [20].

Here more complex setups are going to be implemented and described.

Jetty live WAR

This implementation is made based on the material [20.1]. The following was added:

  • Spring MVC support
  • JSP support
  • necessary resource access restrictions
